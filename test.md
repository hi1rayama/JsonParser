# 構文解析のテスト項目

* NUMBER
  * ~~整数型~~
  * 浮動小数点型
  * 実数
* ~~STRING~~
* ~~True or False~~
* ARRAY
  * 単一要素
    * 数値 [1]
    * 文字列 ["a"]
    * true [true]
    * false
    * null
    * 配列 [[]]
  * 要素なし []
  * 複数要素(オブジェクト)
    * value
      * 整数
      * 文字列
      * true
      * false
      * null
      * オブジェクト
* OBJECT
  * 要素なし
  * 要素あり
    * 配列
    * 整数
    * 文字列
    * true
    * false
    * オブジェクト
